rslidy

  • Drop-in replacement for W3C Slidy and Slidy2.
  • Responsive HTML5, CSS3, JavaScript (TypeScript).
  • No other dependencies.
  • Lightweight: 
    rslidy.js         79,485 bytes
rslidy.min.js     38,412 bytes
rslidy.css        25,456 bytes
rslidy.min.css    17,030 bytes

rslidy Features

  • Slide Viewer:
    • Animated slide transitions.
    • Overview panel.
    • Table of contents panel.
    • Progress bar.
    • Image viewer with pan + zoom.
    • Low light mode.
  • Responsive Layout:
    • Portrait + landscape.
    • Print style sheet.
  • Responsive Interaction:
    • Swipe, pinch, tap.
    • Tilt, shake.
    • Keyboard.
    • Mouse pointer, mouse wheel.
