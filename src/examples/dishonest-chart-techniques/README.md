## External Libraries
Downloaded: May 2026
The following libraries are used locally (stored in the `libs/` folder):
- Chart.js v4.5.1 - https://cdn.jsdelivr.net/npm/chart.js
- D3.js v7.9.0 - https://d3js.org/d3.v7.min.js
- TopoJSON Client v3.1.0 - https://unpkg.com/topojson-client@3

These were downloaded to ensure the slide deck works even when the URLs become unavailable.
