Rslidy

  • Inspired by Slidy and Slidy2
  • Presentation tool using HTML5, CSS3, JavaScript (TypeScript)
  • Runs entirely in the web browser
  • Responsive design with smooth slide transitions
  • Slides are created using <section> or <div class="slide"> blocks
  • Lightweight and dependency-free
